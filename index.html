  5. Para que serve a tag "label"? Para que serve o atributo "for" nessa tag?

    A tag label define um rótulo para muitos elementos de formulário. Esse elemento é útil para usuários de leitor de tela, porque o leitor de tela irá ler em voz alta o rótulo quando o usuário se concentrar no elemento de entrada. Além disso, o elemento label também ajuda usuários que têm dificuldade em clicar em regiões muito pequenas (como botões de rádio ou caixas de seleção) - porque quando o usuário clica no texto dentro do elemento label ele ativa o botão de rádio/caixa de seleção. O atributo for da tag label deve ser igual ao atributo id do elemento input para vinculá-los.

  7. Nas tags de formulário para que serve o atributo "required"?

    O atributo required especifica que um campo de entrada deve ser preenchido antes de enviar o formulário. Campo obrigatório. Funciona com os seguintes tipos de entrada: texto, pesquisa, url, tel, e-mail, senha, selecionadores de data, número, caixa de seleção, rádio e arquivo.

  8. Para que serve a tag "textarea"?

    A tag textarea inclui um texto (múltiplas linhas) dentro do campo. Pode conter um número ilimitado de caracteres e o texto é renderizado em uma fonte de largura fixa (geralmente Courier).

  10. Na tag form para que serve o atributo "method"?

    O atributo method define qual o método HTTP para enviar os dados (ele pode ser "GET" ou "POST")
  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?

    O valor do atributo method na tag form deve ser “POST” para que não apareça a senha na barra de endereços.

  12. Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos?

    O valor do atributo method na tag form para fazer upload de arquivos deve ser o “POST”.

  13. O que as tags fieldset e legend produzem?

    A fieldset tag é usada para agrupar elementos relacionados em um formulário. A fieldset tag desenha uma caixa ao redor dos elementos relacionados. A tag legend é usada para definir uma legenda para o fieldset elemento.
